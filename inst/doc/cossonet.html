Introduction

The package is a integrated function that estimates sparse nonlinear components using COSSO penalty. This package is available from the Comprehensive R Archive Network. This script describes a example of how to use the package.

Installation

We first load the library for and set a seed for reproducibility.

devtools::install_github("jiieunshin/cossonet")
library(cossonet)
set.seed(20250101)

Data generation

The function generates example datasets with continuous response. We generate a training set with \(n=200\) and \(p=20\), and a test set with \(n=1000\) and \(p=20\).

tr = data_generation(n = 200, p = 20, SNR = 9, response = "continuous")

Model fitting

The function is the main function that fits the model. We need to input specific arguments to this function. is the type of response, and and are vector sequences of specific intervals.

lambda0_seq = exp(seq(log(2^{-5}), log(2^{-1}), length.out = 20))
lambda_theta_seq = exp(seq(log(2^{-8}), log(2^{-5}), length.out = 20))

fit = cossonet(tr$x, tr$y, family = 'gaussian',
           lambda0 = lambda0_seq,
           lambda_theta = lambda_theta_seq
           )

When the function is running, the progress at each step is displayed in the console. At each step, cross-validation (CV) is performed and the mean squared error (MSE) of the training set evaluated with the optimal smoothing parameter selected by CV is displayed.

Prediction

The function is used to predict for new data using the fitted model. The output of the function includes the predicted values \(\hat{f}\) from and \(\hat{\mu}\) from for the new data. The predicted value and predictive accuracy for the test set using our fitted model can be obtained by

pred = cossonet.predict(fit, te$x)
str(pred)
